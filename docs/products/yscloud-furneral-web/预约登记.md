### 预约登记

预约登记页面可以填写 **预约登记表、接运登记表**。

在**【预约登记表】** 中，填写是逝者的基础信息，包括性别、业务的类型。

在**【接运登记表】** 中，填写 **接运类型** 和 **接运地址**。
